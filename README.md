# GPS and Fuel Sensor Tracking System

## Project Overview
This project is designed to track vehicle location and monitor fuel levels using GPS and fuel sensors, providing real-time data to improve fuel efficiency and ensure optimal vehicle management.

## Features
- **Real-time GPS Tracking**: Monitors and displays the real-time location of vehicles on a map.
- **Fuel Level Monitoring**: Reports fuel levels directly from the sensors, allowing for accurate fuel management.
- **Alerts**: Notifications for low fuel levels and vehicle movement outside designated areas.
- **Data Logging**: Comprehensive logs of fuel consumption and GPS locations for historical analysis.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ndombele2602/gps-fuel-tracker.git
   cd gps-fuel-tracker
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the application**:
   ```bash
   npm start
   ```
4. **Open your browser** and navigate to `http://localhost:3000` to view the application.
